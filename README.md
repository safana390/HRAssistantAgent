# HR AI Agent


**Short overview**: A conversational HR assistant that answers employee queries (leave policies, payroll, onboarding steps), schedules interviews, and generates template responses. Built with Google AI Studio models for NLU and a Streamlit demo front-end.


## Features
- Natural language Q&A on HR policies
- Leave request handling (template generation)
- Interview scheduling helper (suggests slots)
- Document retrieval from a small knowledge base (PDFs/FAQs)
- Admin panel (view logs and feedback)
